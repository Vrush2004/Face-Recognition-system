About the Project

Welcome to the Face Recognition Attendance Management System! This project is designed to automate attendance using advanced facial recognition techniques. The goal is to eliminate the need for manual attendance-taking and improve accuracy and security for educational institutions, offices, and other organizations.

Project Features:

  • Automated attendance management with real-time facial recognition
  • Efficient data storage and retrieval in a MySQL database
  • Increased security with facial authentication for attendance marking
  • Admin panel for managing attendance records
  • Real-time processing with OpenCV and machine learning models

Technologies Used:

  • Python (OpenCV, TensorFlow, Keras)
  • Pre-trained Models : AlexNet, GoogleNet, SqueezeNet
  • MySQL Database for data storage
  • HTML, CSS, and JavaScript for the front-end
  • Machine Learning for facial recognition and prediction

How It Works:

    The system uses a camera to capture live images of users. The captured images are processed, and the face is detected and recognized using machine learning algorithms. Once recognized, the user is marked as present, and the attendance is saved in the database. The system is easy to use and is efficient in managing the attendance process, reducing human error and improving security.
